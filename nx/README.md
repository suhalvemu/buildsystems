# Nx

> Smart, fast, extensible build system for full-stack monorepos.

---

## What is Nx?

Nx is a build system and monorepo management tool with first-class support for JavaScript, TypeScript, React, Angular, Node, and more. It offers code generation, dependency graph visualization, affected-command detection, and distributed caching.

---

## When to Use Nx

Use Nx when:
- You have a full-stack JS/TS monorepo (frontend + backend)
- You want rich code generation and project scaffolding
- You need affected-build detection (`nx affected`)
- Your team uses Angular (Nx has the best Angular monorepo support)

---

## Nx vs Turborepo

| | Nx | Turborepo |
|---|---|---|
| Code generation | Yes | No |
| Affected detection | Yes (graph-aware) | Yes (basic) |
| Plugin ecosystem | Rich | Minimal |
| Angular support | First-class | Community |
| Learning curve | Higher | Lower |
| Remote cache | Nx Cloud (paid) | Vercel (free tier) |
