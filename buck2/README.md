# Buck2

> Meta's open-source, next-generation build system for large monorepos.

---

## What is Buck2?

Buck2 is Meta's open-source build system, a complete rewrite of the original Buck. It is designed for massive monorepos with strict reproducibility requirements. It uses Starlark (the same language as Bazel) for build rules.

---

## When to Use Buck2

Use Buck2 when:
- You're already in the Meta/Facebook ecosystem
- You need Bazel-like reproducibility with better performance
- You want a modern, actively developed alternative to Bazel

---

## Buck2 vs Bazel

| | Buck2 | Bazel |
|---|---|---|
| Created by | Meta | Google |
| Config language | Starlark | Starlark |
| Remote execution | Yes | Yes |
| Community | Smaller | Larger |
| Rules ecosystem | Growing | Mature |
| Performance | Faster in benchmarks | Battle-tested |
